При запуске ball.exe вас встречает главное меню, 
задний фон которого меняется в зависимости от времени суток - утро, день, вечер и ночь
Есть две опции - для 1 игрока и для 2 игроков, где играют до 10 очков.
Эта игра является результатом изучения tkinter, 2-мерной анимации и
компиляции .py файлов в .exe файлы через auto-py-to-exe
